## Installation

Install `safemocker` as a dev dependency:

<Tabs items={['npm', 'pnpm', 'yarn']} groupId="package-manager" persist>
  <Tab value="npm">
```bash
npm install --save-dev @jsonbored/safemocker
```
  </Tab>
  <Tab value="pnpm">
```bash
pnpm add -D @jsonbored/safemocker
```
  </Tab>
  <Tab value="yarn">
```bash
yarn add -D @jsonbored/safemocker
```
  </Tab>
</Tabs>

## Peer Dependencies

`safemocker` requires the following peer dependencies:

- `next-safe-action`: `^8.0.0`
- `zod`: `^4.0.0`

Make sure these are installed in your project:

```bash
npm install next-safe-action zod
```

## Next Steps

Once installed, follow the [Quick Start Guide](/docs/getting-started/quick-start) to set up safemocker in your project.
